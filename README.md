# 🌍 Country History Explorer

Type any country and travel through **30 defining chapters of its history**, each
shown as a styled "era card" with a date, a vivid scene, and a short summary.

The history is generated by **Claude** (Anthropic's AI). The app is written in
Python using only the standard library — **nothing to install**.

---

## Run it (2 steps)

1. Open the Terminal app and go to this folder:
   ```
   cd ~/Documents/Claude
   ```
2. Start the app:
   ```
   python3 server.py
   ```

Your browser opens automatically at **http://localhost:8000**.
To stop the app, go back to the Terminal and press **Ctrl + C**.

> Without an API key (see below) the app runs in **Demo mode** and always shows a
> built-in sample for Egypt — handy for seeing how it looks.

---

## Turn on real results (add your API key)

To generate a real history for *any* country, you need a Claude API key.

1. Get a key from **https://console.anthropic.com** (Settings → API Keys).
2. In this folder, create a file named exactly `.env` containing one line:
   ```
   ANTHROPIC_API_KEY=sk-ant-...your key here...
   ```
3. Restart the app (`Ctrl + C`, then `python3 server.py` again).

The Terminal will now say **Mode: LIVE**. Type any country and explore!

Your key stays on your computer in `.env`, which is ignored by git and never
shared.

---

## What's in this folder

| File | What it does |
|------|--------------|
| `server.py` | The small web server. Holds your key, talks to the AI. |
| `demo_data.py` | The built-in sample shown in Demo mode. |
| `public/index.html` | The web page. |
| `public/style.css` | How the page looks. |
| `public/app.js` | The page's logic (search + drawing the cards). |
| `.env` | Your secret API key (you create this; not shared). |

---

## Notes

- Each lookup is cached while the app runs, so asking for the same country twice
  is instant and free.
- The history is AI-generated. It's great for exploring, but double-check
  anything important against a trusted source.
